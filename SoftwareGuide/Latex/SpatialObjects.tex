\section{Introduction}
\label{Introduction}

In the previous chapter fundamental data representation types such as
\doxygen{Image} and \doxygen{Mesh} were introduced.  These classes represent
data at a very low-level. Spatial objects organize data into higher-level
abstractions that link data to the anatomical structure from which it was
obtained. In this sense spatial objects are data representational types as
well. In this chapter several examples are used to demonstrate the many
spatial objects found in ITK and how they can be organized into hierarchies
using \doxygen{SceneSpatialObject}. Further the examples illustrate how to use
SpatialObject transformations to control and calculate the position of data.
